# JapaneseChat (Modified by ryoma_1125)

Minecraftサーバーのチャットを日本語（ローマ字変換）にするプラグインです。
このリポジトリは [sya-ri/JapaneseChat](https://github.com/sya-ri/JapaneseChat) をベースに改変したものです。

### 🛠 追加・変更機能
- **チャット変換スキップ機能**: メッセージの先頭に `$` をつけて送信すると、日本語変換をせずにそのまま送信されます。
    - 例: `$10` と打てば、全角の「１０」に変換されず、半角の「10」として送信されます。
- **プラグイン情報の更新**: `/plugins` 等で表示される作者名に `ryoma_1125` を追加しました。

### 📋 基本仕様
- チャットフォーマットやコンフィグなどは実装していません。
- 自由に改変・二次配布が可能です。
